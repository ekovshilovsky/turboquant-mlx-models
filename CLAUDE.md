# TurboQuant-MLX Models — Agent Context

## Overview

Conversion scripts, validation tooling, and HuggingFace model cards for
TurboQuant-compressed model weights. Model weights are stored on HuggingFace
Hub, not in this repo.

## Dependencies

- `turboquant-mlx-core` (tq-convert CLI)
- Python 3.11+ with uv for venv management
- huggingface-cli for model upload
